# Thinc. git workshop

## Activity

- Register github
- Fork this repository
- Clone your version of the repository to your machine
- Create a file named by your nickname
  - Bonus: add extension to indicate your team
    - .kt, .java : Android
    - .html, .js, .css : Web
    - .ts : Backend
    - .php, .ruby, .py, ... : Others
- Inside the file put a message to tell me using any text editor (notepad, vim, vscode, sublime textâ€¦ your choice)
- Commit + Push that file to my repository
- Create a pull request to my repository
